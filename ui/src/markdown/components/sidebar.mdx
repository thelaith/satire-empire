## Usage

First wrap the `SidebarProvider` and `SidebarInset` components around your app.

### page.tsx

```tsx file=<rootDir>/src/examples/ui/sidebar/page.tsx
```

and then add the sidebar component (not components/ui/sidebar.tsx, but sidebar example).
